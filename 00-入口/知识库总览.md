# 知识库总览

标签：#总览 #Obsidian #Docsify #GitHubPages

这个知识库把当前文件夹内的原始资料组织成可编辑、可检索、可发布的 Markdown 资料库。

## 范围

已纳入索引的原始资料包括：

- AI-10、AI-10W 产品规格书。
- FO101-KIT 人脸模组测试板工具板规格书。
- AI-10 SDK 文档、照片下发软件开发手册。
- THFaceCropper SDK、Java JNA 参考包。
- AI-10 调试工具、串口助手、串口驱动。
- 照片下发图片要求说明。
- AI-10 STEP 结构文件压缩包。
- 全部 6 个 PDF 的分页全文 Markdown。

## 非目标

- 不移动、改名或解压原始 PDF、ZIP、RAR 文件。
- 不把压缩包内容直接展开到知识库。
- 不在未读取原文的情况下伪造详细技术参数。

## 阅读路径

1. 先看 [资料地图](资料地图.md)，理解资料之间的关系。
2. 按硬件、软件、工具、照片下发四类进入对应资料卡。
3. 需要检索 PDF 原文时，进入 [PDF 全文索引](/60-PDF全文/全文索引.md)。
4. 需要下载原始文件时，进入 [附件索引](/90-附件/附件索引.md)。
5. 需要发布到线上时，查看 [编辑与发布流程](编辑与发布流程.md)。

## 维护原则

- Markdown 页面负责解释、分类、关联和沉淀决策。
- 原始文件保持原样，作为证据和附件。
- 每个资料卡都链接到原始附件和相关主题。
- PDF 全文由 `scripts/extract-pdf-fulltext.ps1` 生成，不手工编辑生成区正文。
- 新增文件后同步更新 [附件索引](/90-附件/附件索引.md) 和 `_sidebar.md`。
